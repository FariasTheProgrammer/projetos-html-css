<Inputs>

Bem-vindo a 3º Aula de HTML. Nela nós veremos um pouco sobre os mais diversos inputs disponíveis nessa linguagem de marcação.

<Inputs com Texto>

Usando <input> podemos criar campos de preenchimento, com ele podemos receber informações como Nomes, Email, Senhas e muitos outros. Segue uma lista simples com vários tipos de inputs.

<label> Text: <input type="text"></label>
<br> <br>
<label> URL: <input type="url"></label minhna namorada é top >
<br> <br>
<label> E-mail: <input type="email"></label>
<br> <br>
<label> Number: <input type="number"></label>
<br> <br>
<label> Password: <input type="password"></label>
<br> <br>
<label> Tel: <input type="tel"></label>
<br> <br>
<label> Search: <input type="search"></label>
Exemplo 01

O tipo do input irá variar de acordo com o atributo type que você colocar.

  • text: Texto
  • url: Usados para facilitar a identificação de Urls válidos
  • email: Usados para facilitar a identificação de Emails válidos
  • number: Permite apenas a entrada de números
  • password: Esconde o texto
  • tel: Semelhante ao Texto, mas em dispositivos móveis irá abrir o teclado para telefone
  • number: Permite apenas a entrada de números
  • search: Usados para facilitar em pesquisas

<Inputs de Botões>

Estes inputs são botões que podem realizar algo após clicar neles

<form action="">
    <label>File: <input type="file"></label>
    <br> <br>
    <label>Reset: <input type="reset"></label>
    <br> <br>
    <label>Submit: <input type="submit"></label>
    <br> <br>
    <label>Button: <input type="button" value="Botão"></label>
    <br> <br>
    <label>Image: <input type="image" value="Imagem" src="../../img/htmlcss.png"></label>
    <br> <br>
    <label>Color: <input type="color"></label>
    <br> <br>
    <label>Radio: <input type="radio" name="teste">1 <input type="radio" name="teste">2</ label>
    <br> <br>
    <label>Checkbox: <input type="checkbox">1 <input type="checkbox">2</label>
</form>
Exemplo 02

No exemplo acima nós temos uma tag nova, o <form>, ele é usado para defenir um grupo de inputs que serão usados para um formulário, ou seja, um lugar onde podemos pegar as informações forncedidas e reutiliza-las mais para frente.

  • file: Usados para importar arquivos, sendo eles imagens ou outras coisas
  • submit: Usados para concluir o preenchimento do formulário
  • reset: Usados para limpar os campos de um formulário
  • button: Um botão comum, pode ser usado para qualquer ação usando outras linguagens de programação
  • image: Um botão comum, mas como uma imagem para representar
  • color: Usados para permitir que o usuário escolha uma cor
  • radio: Um conjunto de botões que permitem selecionar apenas uma opção
  • checkbox: Um conjunto de botões que permitem selecionar várias opções

No exemplo acima nós temos uma tag nova, o <form>, ele é usado para defenir um grupo de inputs que serão usados para um formulário, ou seja, um lugar onde podemos pegar as informações forncedidas e reutiliza-las mais para frente.

  • date: Usados apenas para pegar a Data
