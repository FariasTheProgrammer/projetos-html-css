Aula 01 - Tags Básicas

Bem-vindo a 1º Aula de HTML. Ela é uma Linguagem de Programação MARCAÇÃO!. Isso mesmo, Marcação!

Diferente do que muitas pessoas pensam, não se programa em HTML ou CSS, pois ela não irá realizar tarefas de rachar a cuca, ela irá defenir marcações para que o navegador entenda de uma maneira, assim, deixando um texto bonito e colorido.

Vamos então começar: Como crio uma página em HTML? É simples, basta ter um editor de texo em sua máquina (Bloco de Notas) e digitar assim:

<!DOCTYPE>
<html lang="pt-BR">
     <head>
         <meta charset="UTF-8">
         <meta name="viewport" content="width=device-width, initial-scale=1.0">
         <title>Minha 1º Página!</title>
     </head>
<body>
         <p>
             Olá Mundo!
         </p>
</body>
</html>

Após digitar tudo isso, deve salvar o documento com a extenção '.html', para fazer isso, vá em 'Salvar Como', digite 'index.html' e pronto! Você tem sua primeira página de HTML funcional! Mas o que esse mundarel de palavras significam???

Tags!

Tags são as marcações no HTML para fazer ele funcionar, para exemplificar, vamos começar de cima pra baixo usando de exemplo o código acima:

  • <!DOCTYPE html>: Define que o Arquivo é um arquivo HTML 5 (versão mais atual do HTML), portanto devemos sempre iniciar o arquivo com essa tag
  • <html>: Esta tag inicia e indica de fato que estamos escrevendo em HTML, tudo que vai dentro dele é HTML puro! Note que temos um lang="pt-BR", isso é um ATRIBUTO, ele seria um "adjetivo", uma informação adicional sobre a tag; neste caso, estamos definindo que o nosso arquivo html está definida como Portugues Brasileiro.
  • <tag>
