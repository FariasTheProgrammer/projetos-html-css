<Display & Tags de Separaçãoe>

Bem-vindo a 4º Aula de HTML. Nela nós veremos um pouco sobre elementos Block & Inline, e sobre algumas tags usadas para facilitar a estrutura do site.

<Elementos Block & Inline>

Falando agora em visual, temos que ter a noção que existem 2 tipos de display nas tags, os Block que ocupam uma "linha" inteira, e os Inline que ocupam apenas o seu tamanho. Para entender melhor, vamos ver o exemplo abaixo (Foi usado CSS para Facilitar o entendimento, mais a frente iremos ver o que é CSS e para que serve)

<h1>Elementos Block</h1>
<p style="background-color: red;">
     Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolorum suscipit ad, vitae est veritatis.
</p>
<ol style="background-color: green;">
    <li>Item</li>
    <li>Item</li>
    <li>Item</li>
</ol>

<br>

<h2>Elementos Inline</h2>
<span style="background-color: red;">
     Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolorum suscipit ad, vitae est veritatis.
</span>
<span style="background-color: green;">
    Lorem ipsum dolor sit amet consectetur adipisicing elit. Dolorum suscipit ad, vitae est veritatis.
</span">
Exemplo 01

<Tags de Agrupamentos>

Agora você verá mais algumas tags úteis para agrupar conteúdos, isso irá ajudar o navegador a defenir o que é importante e ajudará a você a se organizar melhor.

<header>
    <h1>Cabeçalho</h1>
    <nav>
        <ul>
            <li>
                Menu
            </li>
            <li>
                Menu
            </li>
        </ul>
    </nav>
</header>

<main>
    <section>
        Seção 1
        <div>
            Conteúdo
        </div>
    </section>

    <section>
        Seção 2
        <div>
            Conteúdo
        </div>
    </section>

    <section>
        Seção 3
        <div>
            Conteúdo
        </div>
    </section>
</main>

<div>
    Conteúdo
</div>

<aside>
    <nav>
        <ul>
            <li>
                Menu
            </li>
            <li>
                Menu
            </li>
        </ul>
    </nav>
</aside>

<footer>
    <h1>Rodapé</h1>
</footer>
Exemplo 02

Quanta coisa né? Pois é.

Vamos por partes então:

  • <header>: É o Cabeçalho da página
  • <nav>: Navegação, é usada para defenir menus e derivados
  • <main>: Para defenir o corpo Principal se seu conteúdo
  • <section>: Define seções dentro da página
  • <div>: Separação normal, imagine que isto é uma caixa, você pode colocar o que quiser dentro dela
  • <aside>: Indica que algo que está nos cantos das páginas
  • <footer>: É o Rodapé da página

Caso não tenha ficado claro, essas tags são definições que podem ser usadas pelo navegador para definir importâncias. Por exemplo, se o navegador tiver uma função de acessibilidade, ele saberá que deverá começar pela <main>.

<Atributos de Identificação & Classificação>

Vamos agora dar uma breve iniciada na matéria de CSS, portanto vamos aprender um pouco sobre os atributos de id & class

<p id="p1">
    Lorem ipsum dolor sit amet consectetur adipisicing elit.
</p>

<p id="p2" class="importante">
    Lorem ipsum dolor sit amet consectetur adipisicing elit.
</p>

<p id="p3" class="importante">
    Lorem ipsum dolor sit amet consectetur adipisicing elit.
</p>
Exemplo 03

O Atributo de id é um atributo global, todas as tags podem ter, no entanto ela é um atributo de Identificação então ela não pode repetir. Diferente do id, a class pode ser repetida quantas vezes quiser, por ser uma Classificação.
