<CSS>

Bem-vindo a 6º Aula e Última Aula de HTML. Nela nós veremos um pouco sobre CSS.

Quando se trata de CSS temos três opções:

  • Fazer uma Folha de Estilo
  • Usar no mesmo Arquivo
  • Usar nas Tags

<head>
    <link rel="stylesheet" href="style.css">

    <style>
    </style>
</head>

<body>
    <p style=" ">

    </p>
</body>

Dentro de <head> podemos usar um <link> para referenciar um arquivo com extensão .css para ser a nossa Folha de Estilo, ou podemos usar a tag <style> para não precisar criar uma arquivo separado.

Outra opção é usar o atributo style em cada tag que você quiser usar o CSS assim como o <p>

<Seletores>

Vamos ver agora o corpo de um Seletor em CSS:

h1{
    color: blue;
    font-size: 12px;
}
tag/id/classe{
    propriedade: valor;
}

O Esqueleto é como citado acima, primeiro a tag/id/classe e depois suas propriedades e valores.

Essa primeira parte do esqueleto é chamada de Seletor, e é como iremos identificar o que terá o estilo modificado. Vamos ver agora um exemplo prático:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemplo 01</title>

    <style>
        p{
            color: green;
        }

        #p1{
            color: blue;
        }

        .importante{
            color: red;
        }
    </style>
</head>
<body>
    <p id="p1" class="importante">
        Lorem ipsum dolor sit amet consectetur adipisicing elit.
    </p>

    
    <p id="p2" class="importante">
        Lorem ipsum dolor sit amet consectetur adipisicing elit.
    </p>
    
    <p id="p3" class="importante">
        Lorem ipsum dolor sit amet consectetur adipisicing elit.
    </p>
    
    <p id="p4">
        Lorem ipsum dolor sit amet consectetur adipisicing elit.
    </p>
</body>
</html>
Exemplo 01

CSS usa um modelo cascata, ou seja, as propriedades vão adicionando e se sobrepondo de acordo com sua especificidade.
Repare que temos um . e um # nos seletores, o . significa id enquanto o # representa class

Em <style>, definimos que todo <p> será verde (Genérico), mas que todo parágrafo que tiver a Classificação importante terá a cor vermelha (Mediano), e mesmo assim o primeiro parágrafo tem a cor azul, pois definimos que o elemento de id p1 terá a cor azul (Específico).

Então em ordem de importância e especificidade: ID > CLASS > TAG
