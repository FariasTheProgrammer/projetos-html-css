<Listas>

Bem-vindo a 2º Aula de HTML. Nela nós veremos um pouco sobre listas, links e imagens.

Primeiramente pelo começo, vamos iniciar pelo início:
Podemos usar as tags <ol> e <ul> (Ordenadas e Desordenadas respectivamente) para começar a lista, e então, usar a tag <li> (Item de Lista) para cada linha que você desejar. Vejamos um Exmplo:

<ul>
    <li> Item </li>
    <li> Item </li>
    <li> Item </li>
</ul>

<ol>
    <li> Item </li>
    <li> Item </li>
    <li> Item </li>
</ol>
Exemplo 01

Como visto, os <ol> apresentaram os itens com um número atrás indicando seu índice, enquanto as <ul> apenas colocaram uma "bolinha", como se fossem tópicos

<Aninhamento de Listas>

Podemos introduzir nas listas mais listas, assim, produziremos um aninhamento de listas:

<ul>
    <li>
        <ol>
            <li> Item </li>
            <li> Item </li>
            <li> Item </li>
        </ol>
    </li>
    <li>Item </li>
    <li>Item </li>
    <li>Item </li>
</ul>
Exemplo 02

<Lista de Definição>

Estas listas são um pouco diferentes das demais, vamos utilizar para elas as tags <dl> (Lista de Definição), <dt> (Termo da Deninição) & <dd> (Definição). Levando em conta que em uma <dt> pode-se ter mais de uma <dd>.

<dl>
    <dt> Item 1 </dt>
    <dd> Definição 1 </dd>
    <dt> Item 2 </dt>
    <dd> Definição 2.1 </dd>
    <dd> Definição 2.2 </dd>
</dl>
Exemplo 03

<Links>

Os famosos link. Imagino que você já esteja acostumado com eles, ou pelo menos saiba o que são. Em HTML os links são definidos por <a>, e nele deve-se colocar um atributo chamado href para que assim poçamos colocar um link de um site ou parte da página.

Vamos para os Exemplos:

<a href="https://phstefen.github.io/" target="_blank"> Página Incrível do Ph </a>
<br>
<a href="mailto:ph.stefen@gmail.com"> Enviar um E-mail para mim </a>
Exemplo 04

Repare que no primeiro link temos um atributo target, ele serve para definir se o link irá abrir uma página na guia atual ou em uma nova, no exemplo ele abrirá em uma nova guia, para abrir na atual é só não utilizar esse atributo.

<Imagens>

Usando as tags <img> podemos adicionar imagens na página, usando links de imagens ou usando imagens que estão em pastas diferentes do projeto. Para definir devemos usar o atributo src

<img src="https://cdn.mos.cms.futurecdn.net/hFm4iWXhbw4c4rdcMH8tUD.jpg" height="100">
<br>
<img src="../img/unicesumar.png" height="100">
