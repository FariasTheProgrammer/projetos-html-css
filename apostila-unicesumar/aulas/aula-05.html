<Tabelas>

As Tabelas foram tags muito utilizadas antigamentes para fazer o layout de sites, hoje em dia é um método considerado ultrapassado e não é muito bom para o usuário, principalmente para usuários mobile. Mas ainda é importante aprender para usar como tabelas mesmo.

Para criar uma tabela usamos a tag <table>, dentro dela iremos usar as tags <thead>, <tbody>, <tfoot>, <tr>, <td> & <th>. Vamos fazer um Exemplo:

<table border="1">
    <thead>
        <th>Coluna 1</th>
        <th>Coluna 2</th>
    </thead>

    <tbody>
        <tr>
            <td>Linha 1 - Coluna 1</td>
            <td>Linha 1 - Coluna 2</td>
        </tr>
        <tr>
            <td>Linha 2 - Coluna 1</td>
            <td>Linha 2 - Coluna 2</td>
        </tr>
    </tbody>

    <tfoot>
        <th>Rodapé 1</th>
        <th>Rodapé 2</th>
    </tfoot>
</table>
Exemplo 01

Repare que em <table> temos um atributo border, ela define o tamanho da borda em pixels.

<Formulários>

Como visto previamente e bem rapidamente, formulários servem para receber as informações que os usuários preenchem em uma página, no entanto para utiliza-los corretamente devmos conhecer os seguintes atributos: method & action.

<h4>Método GET</h4>
<form action="teste.html" method="GET">
    <label>Nome: <input type="text" placeholder="Insira seu Nome"></label>
    <input type="submit" value="Enviar">
</form>

<br>

<h4>Método POST</h4>
<form action="teste.html" method="POST">
    <label>Nome: <input type="text" placeholder="Insira seu Nome"></label>
    <br>
    <label>Senha: <input type="password" placeholder="Insira sua Senha"></label>
    <input type="submit" value="Logar">
</form>
Exemplo 01

O Atributo method serve para definirmos como os valores dos inputs passarão para a página que aparece em action. O GET deixa os valores visíveis na URL então não é recomendado para uma tela de Login por exemplo. Nestes casos, onde é importante ter a segurança do usuário, usamos o método POST, onde os dados não são visíveis na URL.

<Atributos de Identificação & Classificação>

Vamos agora dar uma breve iniciada na matéria de CSS, portanto vamos aprender um pouco sobre os atributos de id & class

<p id="p1">
    Lorem ipsum dolor sit amet consectetur adipisicing elit.
</p>

<p id="p2" class="importante">
    Lorem ipsum dolor sit amet consectetur adipisicing elit.
</p>

<p id="p3" class="importante">
    Lorem ipsum dolor sit amet consectetur adipisicing elit.
</p>
Exemplo 03

O Atributo de id é um atributo global, todas as tags podem ter, no entanto ela é um atributo de Identificação então ela não pode repetir. Diferente do id, a class pode ser repetida quantas vezes quiser, por ser uma Classificação.
